# Portfolio UX/UI – Faniry

Bienvenue sur mon portfolio de designer UX/UI, inspiré de l'univers de Red Dead Redemption 2 🎮🤠  
Ce site a été entièrement codé en HTML & CSS, avec un effet de scroll immersif, un formulaire de contact fonctionnel et une ambiance western moderne.

## 🔥 Fonctionnalités

- Design responsive (mobile & desktop)
- Effets de scroll cinématographiques
- Menu burger animé sur mobile
- Formulaire de contact avec [Formsubmit](https://formsubmit.co)
- Hébergement gratuit via GitHub Pages

## 🛠 Technologies utilisées

- HTML5 / CSS3
- Google Fonts (Cinzel + Inter)
- Formsubmit
- GitHub Pages

## 💡 Pourquoi ce projet ?

Je suis actuellement à la recherche d’un contrat d’alternance UX/UI.  
Ce portfolio me permet de présenter mon univers, mes projets, ma personnalité et ma vision du design.
